🔧 6. THE FIX PLAN — Exactly What Changes in Each Variant

🔑 Core Fix: Glassmorphism on "GlassBox AI" Text

The text sits inside a frosted glass panel:

  • Background: rgba(255,255,255,0.15-0.25) — semi-transparent white
  • Backdrop-filter: blur(12px) — frosted glass effect
  • Border: 1px solid rgba(255,255,255,0.3) — glass edge catching light
  • Border-radius: 8-12px — smooth glass panel
  • Box-shadow: inset 0 1px 0 rgba(255,255,255,0.2) — top highlight like real glass
  • Text: White, 900 weight, letter-spacing, bigger size

🔑 Core Fix: GitHub Repo Link as Glass Pill Badge

Not small grey text. A highlighted glass pill:

  • Semi-transparent white pill shape with blur
  • ★ github.com/agentic-trust-labs/glassbox-ai in bold
  • Bigger font size, prominent placement

🔑 Core Fix: Personal Side — No QR, Filled With Content

Without QR code, fill the space with:

  • "10 years in AI, ML & everything Agentic" — experience line
